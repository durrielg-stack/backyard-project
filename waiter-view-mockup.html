Planning Notes
What waiters CAN do
View all tables and their status
Open any table's running order
Add new items to any table
Add item-level notes (e.g. "no chili")
See running totals (read-only)
What waiters CANNOT do
Void or cancel items (manager only)
Process payments or close orders
Access reports or budget
See owner/manager tools
Delete or modify past items
DB Changes Needed
Add 'waiter' to users.role CHECK constraint
Create waiter accounts (same pattern as owners)
RLS: waiters same as current 'staff' permissions
No schema changes to orders/items tables

Detect role on login → redirect to /waiter

Mobile-only layout (no 1920px shell)
